Blockquotes

Blockquotes are mainly used to give emphasis to a quote or citation. You can also use these for some extra text hierarchy and emphasis.

This is an example quotation that uses the blockquote tag.
Here is another line to make it look bigger. 
<blockquote>
This is an example quotation that uses the blockquote tag.
</blockquote>

Quote marks

Quote marks adds stylized quote marks before and after an element. Combine with other typography classes to achieve a pull quote effect.

This is an example quotation that uses the blockquote tag.
Here is another line to make it look bigger. 

<div class="quote-marks">
            <h3 class="text-center">             
              This is an example quotation that uses the blockquote tag.
              <br>Here is another line to make it look bigger.
            </h3>
          </div>
